
ฝึกทบทวนความรู้ JavaScript ที่เรียนมาปีก่อน โดยใช้โจทย์ฝึกเขียนฟังก์ชัน 15 ข้อ ครอบคลุมทั้งการใช้

* การวนลูป (`for`, `while`, `for...of`)
* การใช้เมธอดจัดการอาเรย์ (`map`, `filter`, `reduce`, `sort`, `join`)
* การจัดการข้อมูลตัวเลข, string, object, และ array ของ object
* การประยุกต์ใช้เงื่อนไข (`if...else`, ternary operator)

**วัตถุประสงค์:**

* ทบทวนพื้นฐานการเขียนฟังก์ชันใน JavaScript
* ฝึกคิดเชิงตรรกะและการแก้ปัญหาขั้นตอนต่อขั้นตอน
* ใช้ทั้งวิธี imperative (เขียนวนลูปเอง) และ functional (ใช้เมธอดของ array) เพื่อเปรียบเทียบความเข้าใจ
